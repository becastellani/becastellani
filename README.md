## 👋 Bernardo Castellani
### Full Stack Developer · PHP & Node.js · Fintech & Microservices

I build and maintain systems where downtime and bugs cost real money. Currently working at **Sicredi Vale do Piquiri**, a credit cooperative, developing microservices for debt collection and financial reconciliation — PHP orchestrated in Kubernetes, integrated with core banking and credit bureaus.

Before that, spent ~2 years at **Alow** modernizing legacy ERPs in Laravel/Symfony: cut API response times by 30% with Redis caching, replaced ~20h/week of manual work with Puppeteer/Selenium automation, and implemented async processing with RabbitMQ for payment and email workflows.

I care about production stability, clean integrations, and solving problems that actually matter.
